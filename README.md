# Gostat
Gostat can show vmstat gragh on browser for Chartjs.  
Buff, free and cpu id view gragh from 10 minutes ago to now.  
Update interval is 1 minutes.

# Run
./gostat   
Default port is 8080.  
Start api server on localhost:8080/gostat.
